# Nihongo Memory Coach

App HTML tĩnh để học JLPT N3/N2 trên GitHub Pages.

## Cách dùng

1. Tạo repository mới trên GitHub.
2. Upload `index.html` và `.nojekyll` vào root của repository.
3. Vào **Settings → Pages**.
4. Chọn **Deploy from a branch**.
5. Chọn branch `main`, folder `/root`, rồi Save.
6. Mở link GitHub Pages bằng điện thoại hoặc máy tính.

## Ghi chú

- App là 1 file HTML độc lập, dữ liệu học đã nhúng sẵn.
- Tiến độ học lưu trong trình duyệt bằng localStorage. Nếu đổi điện thoại/trình duyệt, hãy dùng nút **Xuất tiến độ** rồi nhập lại ở máy mới.
- Có thể dùng trên điện thoại: bố cục responsive, nút đủ lớn, input không bị zoom trên iPhone.
